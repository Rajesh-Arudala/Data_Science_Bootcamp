# Data_Science_Bootcamp
This certificate recognizes the successful completion of the Data Science Bootcamp at OdinSchool, demonstrating hands-on training in core data science concepts and tools. The program covered Python for Data Analysis, Statistics, Exploratory Data Analysis (EDA), Data Cleaning, Data Visualization, and Machine Learning fundamentals. Through practical assignments and real-world projects, the bootcamp strengthened analytical thinking, problem-solving skills, and the ability to extract actionable insights from data, preparing for entry-level roles in data science and analytics.
